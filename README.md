# Coin Change Algorithm Analysis

Comparative analysis of algorithmic approaches to the coin change optimization problem, including brute force, divide-and-conquer, greedy algorithms, and dynamic programming strategies.

This project evaluates the tradeoffs between different algorithmic techniques, including correctness, computational complexity, and performance characteristics.

## Technologies

- C++
- Algorithms
- Data Structures
- Dynamic Programming
- Algorithm Analysis

## Overview

This project was completed as part of an undergraduate computer science algorithms course and involved implementing and comparing multiple approaches to solving the coin change problem.

The goal was to understand how algorithm design choices affect performance and scalability across different problem-solving strategies.
